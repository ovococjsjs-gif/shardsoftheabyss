# Archive

Здесь лежат:
- старые исходные рабочие документы, из которых была собрана новая структура;
- полные дубли файлов.

Архив не используется как обязательный ежедневный пакет.  
Поднимать его только если нужно:
- сверить старую формулировку;
- проверить, не потерялось ли что-то при объединении;
- восстановить историю рабочих решений.
