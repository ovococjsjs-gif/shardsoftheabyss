## Правило работы с аркой 1

Перед любым решением о переписи сначала проводится аудит:
- голос;
- плотность;
- различимость персонажей;
- структурная причинность;
- степень служения плану;
- качество отдельных сцен и переходов.

Только после этого выбирается один из режимов:
- точечные правки;
- частичная перепись сцен;
- полная перепись, если арка реально не выполняет задачу.

## Принцип

Не ломать найденное вслепую.
