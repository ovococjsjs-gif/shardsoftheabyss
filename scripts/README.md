# Scripts

## `style_guard.py`
Жёсткий автоматический guard художественной прозы.

### Что делает
Ищет блокеры проекта:
- отрицательные контрастные конструкции;
- запрещённые словесные маркеры;
- кандидаты на тройные кластеры;
- плотность `будто` / `словно`;
- одиночные слова-абзацы.

### Как запускать
```bash
python3 scripts/style_guard.py <файл>
```

Пример:
```bash
python3 scripts/style_guard.py 03-manuscript/ch-05-draft.md
```

### Правило
Если guard находит `BLOCKER`, текст нельзя показывать как готовый.

### Важно
Проверка троек эвристическая. Она сделана специально жёсткой, но всё равно требует ручного подтверждения автора/редактора.
