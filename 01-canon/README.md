# Canon

Длинная каноническая база проекта.

## Приоритет внутри папки
1. `project-bible.md`
2. `PROJECT-STATUS.md`
3. `world-expansion-v0.1.md`

## Как использовать
- `project-bible.md` — главный источник канона
- `PROJECT-STATUS.md` — краткий текущий статус и решения
- `world-expansion-v0.1.md` — расширение мира и рабочие предложения

Эту папку не нужно перечитывать целиком перед каждой задачей.  
Сначала использовать короткий пакет из `00-start-here/` и `02-writing-system/`.
